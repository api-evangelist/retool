Retool 4.0 is the most significant infrastructure change to self-hosted Retool since launch. It introduces the new app builder built on React, with AI-assisted building and a real-time collaborative editing environment. These features are powered by a set of additional services: the agent sandbox, JS executor, MCP server, and app building agent.
